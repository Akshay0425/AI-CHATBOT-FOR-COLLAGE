    "WHAT ARE THE COURCES AVALIABLE?"
    "TELL ME ABOUT PLACEMENTS"
    "WHAT IS THE FEE STRUCTURE?"
*FEACTURES OF THE CHATBOT
   24/7 AVALIBALE
   INSTANT RESPONSE SYSTEM
   USER-FRIENDLY INTERFACE
   PLACEMENT AND EVENTS UPDATES
*TECHNOLOIGES USED

  AI MODEL(BRAIN):GPT-5 OR CLAUDE 3.5 SONNET
  KNOWLEDGE BASE:SUPABASE(VECTOR STORE)
